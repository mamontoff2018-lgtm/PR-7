### Ответы на контрольные вопросы

**1. Как в Android реализуется локализация приложений? Опишите структуру папок для поддержки нескольких языков.**

Локализация реализуется путём создания альтернативных папок с ресурсами, имеющих квалификатор локали. Структура:
- `res/values/` — ресурсы по умолчанию
- `res/values-ru/` — ресурсы для русского языка
- `res/values-cs/` — ресурсы для чешского языка
- `res/values-en/` — ресурсы для английского языка

В этих папках создаётся файл `strings.xml` с одинаковыми именами ресурсов, но разными значениями.

**2. Для чего используются адаптеры (например, ArrayAdapter) при работе со списками ListView и Spinner?**

Адаптеры служат мостом между источником данных (массив, список, курсор БД) и UI-компонентом (ListView, Spinner). Они отвечают за создание представления для каждого элемента и переиспользование готовых View для оптимизации.

**3. Какие атрибуты EditText позволяют ограничить тип вводимых данных? Приведите примеры.**

Атрибут `android:inputType`:
- `text` — обычный текст
- `textPersonName` — имя человека
- `textEmailAddress` — email
- `textPassword` — пароль (скрытые символы)
- `phone` — телефонный номер
- `number` — целое число
- `date` — дата

**4. Что такое регулярные выражения? Как с их помощью проверить, что строка является валидным email-адресом?**

Регулярные выражения — это шаблоны для поиска и проверки строк. Для проверки email используется выражение:
```
^[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}$
```
Метод проверки: `email.matches(regex)` возвращает true, если строка соответствует шаблону.

**5. Как программно установить ошибку на поле ввода (EditText), чтобы она отображалась пользователю?**

Используется метод `setError(String message)`:
```kotlin
editText.error = "Сообщение об ошибке"
editText.requestFocus()
```

**6. В чём разница между CheckBox и RadioGroup? В каких случаях используется каждый из них?**

- **CheckBox** — позволяет выбрать несколько вариантов (множественный выбор)
- **RadioGroup** — группа RadioButton, позволяющая выбрать только один вариант (единственный выбор)

**7. Как вывести диалоговое окно для выбора даты (DatePickerDialog) и получить выбранное значение?**

```kotlin
DatePickerDialog(this,
    { _, year, month, dayOfMonth ->
        val date = "$dayOfMonth.${month + 1}.$year"
        editText.setText(date)
    },
    year, month, day
).show()
```

**8. Для чего используется метод String.matches()? Что он возвращает?**

Метод `String.matches(String regex)` проверяет, соответствует ли вся строка заданному регулярному выражению. Возвращает `true`, если строка полностью соответствует шаблону, и `false` — если не соответствует.
